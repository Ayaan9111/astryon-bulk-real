WHAT ASTRYÓN IS

Astryón is an AI-powered SaaS tool designed to help real estate professionals generate large volumes of property listing descriptions automatically.

Real estate agents often spend hours writing listing descriptions for property portals, marketing websites, and CRM systems.

Astryón automates this process.

Users upload a spreadsheet of property data and the system generates professional listing descriptions in bulk using AI.

The output is formatted and ready for publishing on property portals or CRM systems.

The goal is to save agencies time and maintain consistent listing quality.

WHAT THE APPLICATION DOES

The application performs the following core functions:

1️⃣ User authentication system
Users can create accounts, login, and manage their subscription.

2️⃣ Dashboard
After login, users access a dashboard showing:

subscription plan

credits remaining

generation history

account management

3️⃣ CSV Upload System
Users upload a CSV containing property data.

Expected columns may include:

property_title

property_type

bedrooms

bathrooms

area

location

price

amenities

nearby_landmarks

additional_notes

The system must parse each row as a separate property listing.

4️⃣ AI Generation Engine

For each property row, the system generates:

• Long professional listing description
• Short portal-ready description
• Optional social media caption

The AI must produce factual structured output without exaggeration or storytelling.

Two output modes must exist:

Concise
Short, structured professional paragraph.

Detailed
Longer, descriptive but still professional.

Each generated listing consumes 1 credit.

5️⃣ Bulk Processing

Users can generate descriptions for many listings at once.

Example:

CSV with 50 rows → generates 50 listings → consumes 50 credits.

6️⃣ Export System

Users can download results as:

CSV export containing generated descriptions.

7️⃣ Credit System

Credits reset monthly based on subscription tier.

200 credits

HOW USERS USE THE PRODUCT

User flow must work like this:

Step 1
User signs up and logs into Astryón.

Step 2
User selects a subscription plan and completes checkout through Paddle.

Step 3
User enters the dashboard.

Step 4
User uploads a CSV containing property data.

Step 5
User selects output mode (Concise or Detailed).

Step 6
User clicks Generate.

Step 7
AI processes the CSV rows and generates descriptions.

Step 8
User downloads generated listings as CSV.

Credits decrease automatically.

AI PROVIDER CONFIGURATION

The AI generation system must be modular.

Create an AI service module that supports multiple providers.

Supported providers:

Primary provider: Groq
Optional upgrade provider: OpenAI

The provider must be controlled by environment variables.

Example environment variables:

GROQ_API_KEY=
OPENAI_API_KEY=
AI_MODEL_NAME=

All generation must happen server-side.

Never expose API keys in frontend code.
